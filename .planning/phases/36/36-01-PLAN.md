    ```python
    class EscalationRequest(Message):
        """An agent escalation question ready for the approval modal."""
        def __init__(self, question: str, agent_id: str = "") -> None:
            self.question = question
            self.agent_id = agent_id
            super().__init__()
    ```

    **2. Create modals.py with three ModalScreen subclasses:**

    `FileApprovalModal(ModalScreen[bool])`:
    - Constructor takes `file_path: str`
    - compose() yields a Grid#dialog containing: Label with "Approve write to:\n{file_path}", Button#approve (variant="success"), Button#deny (variant="error")
    - on_button_pressed: `self.dismiss(event.button.id == "approve")`
    - BINDINGS: `("escape", "cancel", "Cancel")` -> `action_cancel` calls `self.dismiss(False)`
    - DEFAULT_CSS: align center middle, #dialog with width 60, height auto, border solid $primary, background $surface, padding 1 2, .buttons with layout horizontal

    `CommandApprovalModal(ModalScreen[bool])`:
    - Same structure as FileApprovalModal but with `command: str` and label text "Approve command execution?\n{command}"
    - Same button/binding behavior

    `EscalationModal(ModalScreen[str])`:
    - Constructor takes `question: str, agent_id: str = ""`
    - compose() yields Grid#dialog containing: Label with "[{agent_id}] {question}" (prefix only if agent_id non-empty), Input#reply-input (placeholder="Your reply..."), Button#submit (variant="primary")
    - on_button_pressed: get Input value, dismiss with value.strip() or "proceed" if empty
    - on_input_submitted: dismiss with event.value.strip() or "proceed" if empty
    - BINDINGS: escape -> `self.dismiss("proceed")`
