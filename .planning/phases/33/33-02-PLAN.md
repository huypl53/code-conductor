SDK streaming pattern (from chat.py lines 402-462):
```python
from claude_agent_sdk import ClaudeAgentOptions, ClaudeSDKClient, ResultMessage
from claude_agent_sdk.types import StreamEvent

options = ClaudeAgentOptions(
    cwd=cwd,
    permission_mode="bypassPermissions",
    include_partial_messages=True,  # CRITICAL for StreamEvent
    system_prompt={"type": "preset", "preset": "claude_code"},
)
client = ClaudeSDKClient(options=options)
await client.connect()

await client.query(text)
async for message in client.receive_response():
    if isinstance(message, StreamEvent):
        event = message.event
        if event.get("type") == "content_block_delta":
            delta = event.get("delta", {})
            if delta.get("type") == "text_delta":
                chunk = delta.get("text", "")
    elif isinstance(message, ResultMessage):
        usage = message.usage  # dict with input_tokens, output_tokens
```

    ```python
    async def on_user_submitted(self, event: UserSubmitted) -> None:
        pane = self.query_one(TranscriptPane)
        await pane.add_user_message(event.text)
        # Create streaming AssistantCell
        self._active_cell = await pane.add_assistant_streaming()
        # Disable input
        self.query_one(CommandInput).disabled = True
        # Start SDK streaming worker
        self._stream_response(event.text)
    ```

    **4. Add @work streaming coroutine:**
    ```python
    @work(exclusive=True, exit_on_error=False)
    async def _stream_response(self, text: str) -> None:
    ```
    - Inside try block:
      - Call `await self._ensure_sdk_connected()`
      - Call `await self._sdk_client.query(text)`
      - Iterate `async for message in self._sdk_client.receive_response()`
      - On StreamEvent with content_block_delta + text_delta: extract chunk, if first chunk call `await self._active_cell.start_streaming()`, then `await self._active_cell.append_token(chunk)`

    - In except block: catch Exception, mount an error AssistantCell with the error message (static mode with error text).
    - In finally block: call `await self._active_cell.finalize()` if active_cell exists and is streaming, then `self._active_cell = None`, post `StreamDone()`.

    **5. Add on_stream_done handler:**
    ```python
    def on_stream_done(self, event: StreamDone) -> None:
        cmd = self.query_one(CommandInput)
        cmd.disabled = False
        cmd.query_one(Input).focus()
    ```
