    2. Create `packages/conductor-core/src/conductor/dashboard/watcher.py`:
       - `state_watcher(state_path: Path, ws_manager: ConnectionManager, stop_event: asyncio.Event) -> None` async coroutine
       - Uses `watchfiles.awatch(str(state_path.parent), stop_event=stop_event, debounce=200)` — watch parent dir, not file directly (Pitfall 1 from research: os.replace swaps inodes)
       - Filter changes: only process when `state_path.name` is in changed filenames
       - Read state via `asyncio.to_thread(StateManager(state_path).read_state)` — never block event loop
       - Call `classify_delta(prev, new_state)` and broadcast each event via `ws_manager.broadcast(event.model_dump_json())`
       - Track `prev: ConductorState | None = None` across iterations
       - Wrap read_state in try/except Exception: continue (state file may be mid-write)

    3. Create `packages/conductor-core/src/conductor/dashboard/server.py`:
       - `ConnectionManager` class:
         - `active_connections: list[WebSocket] = []`
         - `async connect(websocket)` — await accept, append
         - `disconnect(websocket)` — remove if present
         - `async broadcast(message: str)` — iterate over `list(self.active_connections)` copy (Pitfall 2), wrap individual send_text in try/except, disconnect dead connections
       - `create_app(state_path: Path) -> FastAPI`:
         - Create `ConnectionManager`, `StateManager(state_path)`, `asyncio.Event()` for stop
         - `lifespan` asynccontextmanager: create_task(state_watcher(...)), yield, stop_event.set(), cancel task, await with suppress CancelledError
         - Store ws_manager and state_manager on `app.state`
         - `GET /state` route: `asyncio.to_thread(state_manager.read_state)` then `state.model_dump(mode="json")`
         - `WebSocket /ws` route: connect, send full current state as first message (Pitfall 3 belt-and-suspenders), then loop `receive_text` with `WebSocketDisconnect` handler
       - Use `from __future__ import annotations` (project convention)

    4. Update `packages/conductor-core/src/conductor/dashboard/__init__.py` to also export `create_app` and `ConnectionManager` from server module.
