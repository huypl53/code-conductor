    - Keep existing DEFAULT_CSS (width: 30, border-left, padding)

    - `on_mount()`: if `self._state_path` is not None, call `self._watch_state(self._state_path)`
    - `_watch_state(state_path)` as `@work(exclusive=True, exit_on_error=False)`:
      - Uses `watchfiles.awatch(str(state_path.parent), debounce=200)` -- same pattern as dashboard/watcher.py
      - Filters changes to only state_path.name
      - Reads state via `asyncio.to_thread(StateManager(state_path).read_state)`
      - Posts `self.post_message(AgentStateUpdated(new_state))`

      - Builds `tasks` dict: task.assigned_agent -> task
      - Removes panels for agents no longer active: `await panel.remove()` for each AgentPanel whose agent_id is not in active
      - Mounts new AgentPanel for new agent IDs not in existing panels
      - Updates existing panels via `panel.update_status(...)`
      - Shows/hides "No agents active" Static: hide when panels exist, show when empty

    **IMPORTANT anti-patterns to avoid:**
    - Do NOT watch state_path directly -- watch state_path.parent (inode swap issue with os.replace)
    - Use `async def on_agent_state_updated` so `await panel.remove()` works (sync handler drops the AwaitRemove)
    - Use `id=f"agent-{agent_id}"` for unambiguous query_one lookups
    - Store data in __init__, render in compose() -- do NOT query_one in __init__
