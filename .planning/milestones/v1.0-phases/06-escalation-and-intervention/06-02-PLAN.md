          async def cancel_agent(self, agent_id: str, corrected_spec: TaskSpec) -> None:
          ```
          - Pop agent_id from _active_tasks, cancel() the task, await it (catching CancelledError)
          - Write FAILED status to state via asyncio.to_thread + mutate
          - Spawn new _run_agent_loop with corrected_spec using self._semaphore
          - Store new task in _active_tasks

          async def pause_for_human_decision(

              timeout: float = 120.0,
          ) -> None:
          ```
          - Look up agent_id in _active_clients, raise EscalationError if not found
          - `await client.interrupt()`
          - Drain `stream_response()` (async for _ in client.stream_response(): pass)

          - `await asyncio.wait_for(human_in.get(), timeout=timeout)` — catch TimeoutError, fallback to "proceed with best judgment"
