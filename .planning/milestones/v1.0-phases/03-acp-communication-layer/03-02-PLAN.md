1. Create `src/conductor/acp/client.py` with:

```python
class ACPClient:
    """Async context manager wrapping ClaudeSDKClient for sub-agent sessions.

    Handles session lifecycle, streaming, and correct options wiring
    including the mandatory PreToolUse keepalive hook.

    Usage::
        handler = PermissionHandler(state_manager=mgr)
        async with ACPClient(
            cwd="/path/to/repo",
            system_prompt="You are Ariel, a backend developer.",
            permission_handler=handler,
        ) as client:
            await client.send("Implement the /api/users endpoint")
            async for message in client.stream_response():
                process(message)
    """

    def __init__(
        self,
        *,
        cwd: str,
        system_prompt: str = "",
        allowed_tools: list[str] | None = None,
        permission_handler: PermissionHandler | None = None,
        max_turns: int = 50,
        setting_sources: list[str] | None = None,
    ) -> None:
        ...
```

Key implementation details:
- Store a `_closed: bool = False` flag. `send()` and `stream_response()` raise `SessionError("Session is closed")` if `_closed` is True.
- In `__init__`, build `ClaudeAgentOptions`:
  - `cwd`, `system_prompt`, `max_turns` passed through
  - `allowed_tools` defaults to `["Read", "Edit", "Write", "Bash", "Glob", "Grep"]`
  - `setting_sources` defaults to `["project"]` (loads CLAUDE.md from repo)
  - `permission_mode="default"`
  - If `permission_handler` is provided: set `can_use_tool=permission_handler.handle` AND register the keepalive hook:
    ```python
    async def _keepalive(input_data, tool_use_id, context):
        return {"continue_": True}

    hooks = {"PreToolUse": [HookMatcher(matcher=None, hooks=[_keepalive])]}
    ```
  - If no permission_handler: no `can_use_tool`, no hooks
- `__aenter__`: create `ClaudeSDKClient(options=self._options)`, call `await self._sdk_client.__aenter__()`, return self
- `__aexit__`: call `await self._sdk_client.__aexit__(...)`, set `_closed = True`
- `send(prompt: str)`: call `await self._sdk_client.query(prompt)`
- `stream_response()`: async generator that yields from `self._sdk_client.receive_response()`
- `interrupt()`: call `await self._sdk_client.interrupt()` (for future use in Phase 6)
