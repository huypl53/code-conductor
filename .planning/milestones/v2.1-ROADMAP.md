# Milestone v2.1: UX Polish

**Status:** ✅ SHIPPED 2026-03-12
**Phases:** 39-42
**Total Plans:** 4

## Overview

Refine the Textual TUI to feel native and polished in the terminal — auto-focus, full alt-screen mode, borderless design, smooth animations, and external editor support — bringing the TUI to parity with OpenAI Codex CLI's terminal integration quality.

## Milestone Summary

**Key Decisions:**
- AUTO_FOCUS = "CommandInput Input" for immediate input focus (Phase 39)
- solid border-left at 40% opacity for subtle accent lines (Phase 40)
- styles.animate() over Widget.animate() for CSS property animation (Phase 41)
- Post EditorContentReady to CommandInput directly — messages bubble UP (Phase 42)

**Issues Resolved:**
- Widget.animate raises AttributeError for CSS properties — use styles.animate instead
- Textual message bubbling means app.post_message won't reach child handlers

None
