## Cross-Phase Integration

**Score:** 34/38 integration points wired

### Issues Found

#### HIGH: CLI cancel/redirect signature mismatch

`input_loop.py` calls `orchestrator.cancel_agent(agent_id)` with one argument, but the method requires `(agent_id, corrected_spec: TaskSpec)`. The `redirect` command passes `new_instructions=` which doesn't exist. Both commands will raise `TypeError` at runtime.

**Fix:** Either give `corrected_spec` a default value in `cancel_agent()`, or construct a `TaskSpec` in the CLI dispatch.

#### MEDIUM: EscalationRouter not connected to ACPClient

`EscalationRouter` is created in `Orchestrator.__init__` but never wired as the `permission_handler` for `ACPClient` sessions. This means `AskUserQuestion` tool calls from sub-agents don't route through the auto/interactive escalation logic.

**Fix:** Create `PermissionHandler(answer_fn=self._escalation_router.resolve)` and pass it to `ACPClient`.

#### MEDIUM: pause_for_human_decision unreachable

The method exists but no CLI command or dashboard action invokes it. COMM-07 is architecturally complete but has no user-facing surface.

**Fix:** Add a `pause` command to CLI and/or a "Pause" action in the dashboard InterventionPanel.

#### LOW: Getting-started guide .env claim

The guide states "Conductor automatically reads `.env` files" but no `python-dotenv` dependency or `load_dotenv()` call exists. Fix: remove the claim or add dotenv support.
